###项目简介
```
由于《斗罗大陆》许多平台的漫画资源都是收费的,不能免费观看.
在查看了漫画台的url后,发现有空可爬,于是对漫画台写了这么一个小的爬虫项目.
```

###项目环境
```
· Windows 10 专业工作站版
· Python 3.6.8
· Scrapy 1.6.0
· Pillow 6.0.0
· Docker 2.0.0.3
· Splash 3.3.1

更多请参考 requirements.txt
```

###运行项目
```
在项目中main.py写了命令行调用,可在PyCharm开发工具的控制台看到debug信息.
```
